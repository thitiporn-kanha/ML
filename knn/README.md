# KNN Template

Usage:

- Place your dataset at `knn/data/data.csv` and ensure it contains the target column named `target` or edit `main.py` to change the target name.
- Run `./main.py` (script is executable if shebang is present).

Outputs:

- `knn/results/results.json` — JSON with accuracy and classification report
- `knn/results/model.joblib` — saved trained model
